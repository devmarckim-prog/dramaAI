# ⚙️ 백엔드 오피스: AI 모델 & 프롬프트 제어 센터 구현 완료

관리자가 서비스의 핵심인 AI 모델과 시스템 프롬프트를 실시간으로 제어할 수 있는 기능을 완성했습니다. 이제 코드 수정 없이도 드라마 기획의 톤과 매너를 자유롭게 바꿀 수 있습니다.

## 🌟 구현된 주요 기능

### 1. 전용 설정 데이터베이스(Supabase) 연동
- [x] `system_settings` 테이블을 구축하여 설정을 클라우드에 영구 보관합니다.
- [x] 서버 재시작 시에도 관리자의 설정이 그대로 유지됩니다.
- [x] 클라우드 연동 실패 시 로컬 `system.json` 파일을 백업으로 사용하는 이중화 구조를 적용했습니다.

### 2. 세밀한 AI 모델 분리 제어
- [x] **Planning Model**: 기획안 로그라인 작성 등에 적합한 가성비 모델(Claude 3.5 Haiku 등) 선택 가능.
- [x] **Writing Model**: 실제 시나리오 집필에 최적화된 고성능 모델(Claude 3.5 Sonnet 등) 선택 가능.
- [x] 각 모델의 특징(속도, 비용, 성능)을 UI에 명시하여 관리자의 판단을 돕습니다.

### 3. 마스터 시스템 프롬프트 에디터
- [x] AI가 따를 전역 규칙과 페르소나를 언제든 수정할 수 있습니다.
- [x] 기존의 하드코딩된 로직을 제거하고, 관리자가 입력한 프롬프트를 100% 신뢰하여 반영하도록 로깅 방식을 개선했습니다.

### 4. 관리자 UI 고도화
- [x] '설정' 탭의 디자인을 글래스모피즘 스타일로 개편하여 시각적 완성도를 높였습니다.
- [x] 저장 시 실시간으로 시스템에 반영되며, 대본 생성 로그에서 변경된 설정을 즉시 확인할 수 있습니다.
