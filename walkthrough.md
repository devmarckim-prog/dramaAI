# ⚙️ 백엔드 오피스: AI 모델 & 프롬프트 제어 센터 구현 완료

관리자가 서비스의 핵심인 AI 모델과 시스템 프롬프트를 실시간으로 제어할 수 있는 기능을 완성했습니다. 이제 코드 수정 없이도 드라마 기획의 톤과 매너를 자유롭게 바꿀 수 있습니다.

## 🛠️ 주요 변경 사항

### 1. 관리자 대시보드 강화 (Admin Dashboard Enhancements)
- **샘플 프로젝트 노출 제어**: 관리자 페이지의 '샘플 프로젝트 관리' 탭에서 각 샘플의 **노출/비노출(Exposed/Hidden)** 상태를 즉시 전환할 수 있는 토글 스위치를 추가했습니다.
- **실시간 프로그레스 바**: 모든 프로젝트(샘플 및 일반 프로젝트)의 AI 생성 진행률(`pct`)을 시각화하는 프리미엄 프로그레스 바를 관리자 리스트에 도입했습니다.
- **생성 일시 표시**: 프로젝트가 언제 생성되었는지 한눈에 확인할 수 있도록 포맷팅된 날짜 컬럼을 추가했습니다.
- **공개 API 필터링**: 비노출로 설정된 샘플은 일반 사용자용 메인 페이지(`api/samples`)에서 자동으로 제외되도록 백엔드 로직을 강화했습니다.

### 2. 사용자 경험(UX) 개선
- **로그라인 디폴트 문구**: 프로젝트 생성 위저드(Step 3)의 로그라인 입력창에 이순신 장군 배경의 예시 문구를 기본값으로 설정했습니다.
- **자동 초기화**: 사용자가 입력창을 클릭(Focus)하면 예시 문구가 자동으로 사라져 타이핑이 편리하도록 개선했습니다.

### 3. Vercel 서버리스 환경 안정화
- **파일시스템 오류 해결**: Vercel의 읽기 전용 파일시스템에서 발생하던 `fs.appendFileSync` 로그 기록 오류를 방어적 프로그래밍(Try-Catch)으로 해결했습니다.
- **Supabase 클라이언트 강화**: 환경 변수 미설정 시에도 애플리케이션이 크래시되지 않도록 클라이언트 초기화 로직을 보강했습니다.
